近年来，呵呵/i随着大数据相关的各种技术的迅速发展和普及， 我们已经生活在了一个大数据时代，本项目主要针对商业方面的大数据的采集与分析。 首先使用wifi探针来收集商业门店周围的手机的MAC地址、 出现时间及地点、与探针距离等数据，再将数据定时传到服务端， 同时对数据进行初步处理，再通过storm分布式系统进行实时分析计算， 通过hadoop进行离线分析计算，最后将分析过的数据通过echarts API可视化地从web前端进行展示。 商家可以直观地根据分析结果调整自己的商业策略，制定活动方案， 从而增加商家店铺的吸引力和营销量。
